## 🎨 Aesthetic & Identity
QRky isn't just a generator; it's a **design system**. Built for designers and developers who value privacy and premium aesthetics, QRky allows you to craft QR codes that match your brand's unique DNA.

- **💎 Glassmorphism UI**: A stunning, modern interface inspired by high-end OS designs.
- **🛸 Cyber-Futuristic Styles**: Advanced patterns and granular control over every vertex.
- **🔒 Local-First Privacy**: No data ever leaves your device. No cloud. No tracking.
- **⚡ Real-Time Engine**: Instant, high-performance rendering as you tweak your values.

## ⚡ Deployment & Setup

### Rapid Install
```bash
git clone https://github.com/ajbatac/qrky.git
cd qrky
npm install
npm run dev
```

### Hosting on Netlify
QRky is **Netlify-ready**. Simply push your repository and the included `_redirects` will handle the rest.
